# HTML

É uma linguagem de marcação que tem as seguintes responsabilidades:

- Conteúdo;
- Semanticidade (uso das tags de forma correta);
- Tags de estrutura;

## Tags

`<h1></h1>` - heading 1 - 6
Para definição de títulos.

`<a href=""></a>`- anchor
Usada para definir a navegação do usuário.
